# simple_serverless_lamba

##### Useful commands

 1. Install serverless

    ```
    npm install -g serverless.
    ```
 
 2. Create a new serverless project

    ```  	
    serverless
    ```

 3. Move into the newly created directory

    ```  	
    cd your-service-name
    ```
    
 4. Deploy Serverless

    ```  	
    serverless deploy
    ```
    
  5. Remove serverless

    ```  	
    serverless remove
    ```
